# Panchang - বৈদিক পঞ্জিকা

<img src="asset/logo.png" alt="Panchang Logo" width="100">

**Panchang** is a modern web-based Vedic calendar tool designed for Bengali users. It provides daily astrological and calendrical information including **Tithi, Nakshatra, Yoga, Karana, Bengali date, Sunrise, Sunset**, and more, with an intuitive interface.

This project is perfect for students, devotees, and anyone interested in following the traditional Vedic calendar and Hindu astrology in a user-friendly web format.

---

## 🔗 Live Demo

You can view the live version here: [Panchang Live](https://rantusamadder.github.io/Panchang/)

---

## 📝 Features

* **Daily Panchang Calculation:** Automatically calculates Tithi, Nakshatra, Yoga, and Karana.
* **Bengali Date Conversion:** Converts Gregorian dates to Bengali calendar dates.
* **Sunrise & Sunset:** Shows approximate sunrise and sunset times for Bangladesh.
* **Interactive Input:** Users can select any date and time to calculate Panchang values.
* **Infinite Scroll News Ticker:** Displays a devotional mantra that scrolls infinitely.
* **Responsive Design:** Works on desktop, tablet, and mobile.

## 💻 Technologies Used

* **HTML5 & CSS3**: Semantic markup and modern styling
* **JavaScript (Vanilla)**: Date calculation, Panchang algorithm, dynamic updates

## 🧮 How It Works

1. **User Input:** Select a date and time using the datetime picker.
2. **Panchang Calculations:** JavaScript functions calculate:

   * **Tithi** (Moon phase)
   * **Nakshatra** (Lunar mansion)
   * **Yoga** (Sun-Moon combination)
   * **Karana** (Half Tithi)
   * Bengali date
   * Sunrise & Sunset times
3. **Display Results:** All calculated values are dynamically displayed in the interface.

1. Clone the repository:
